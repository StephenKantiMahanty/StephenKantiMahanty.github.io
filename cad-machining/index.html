Input: DXF export | Output: Drawing + Instructions

// HOW TO USE
  1. In SolidWorks: File → Save As → DXF/DWG (choose flat pattern or drawing view)
  2. Open the saved .dxf file in a text editor, select all, copy
  3. Paste into the box below or use the file upload button
  4. Click Analyze — holes are detected and the drawing is generated
  5. Boundary & edge distances: the dashed rectangle on the drawing marks the part boundary (bounding box of all DXF geometry). Centerlines extend edge-to-edge and are labelled with the distance from each hole centre to that boundary edge. The table shows Edge (ctr) — nearest boundary distance from hole centre — and Wall (min) — remaining material (edge dist − radius). For accurate results, export the part outline too: in SolidWorks flat-pattern DXF export tick Include all sketch entities so the outer profile exports as geometry (LWPOLYLINE/LINE), not just the holes.
